🚀 INICIO RÁPIDO - PROYECTO COLEGIO

═══════════════════════════════════════════════════════════════

📦 PASO 1: Instalar Dependencias
───────────────────────────────────────────────────────────────

cd c:\Users\JERONIMO\Downloads\PROYECTO_COLEGIO
npm install

✅ Esto descargará: express, sqlite3, bcrypt, jsonwebtoken, axios, dotenv


🔑 PASO 2: Configurar Variables de Entorno
───────────────────────────────────────────────────────────────

Abre el archivo .env y configura:

  PORT=3000                                   (puerto del servidor)
  JWT_SECRET=tu_clave_secreta                (para tokens)
  GEMINI_API_KEY=tu_clave_gemini_aqui        (para IA - opcional)
  OPENAI_API_KEY=tu_clave_openai_aqui        (para IA - opcional)

Si no tienes claves de IA, deja los valores vacíos.


▶️ PASO 3: Iniciar el Servidor
───────────────────────────────────────────────────────────────

npm start

Verás algo como:
  ✓ Conectado a la base de datos SQLite en ./data/school.db
  ✓ Servidor escuchando en http://localhost:3000


🌐 PASO 4: Abrir en Navegador
───────────────────────────────────────────────────────────────

http://localhost:3000

Verás la página de LOGIN con dos pestañas:
  👨‍🏫 Profesor
  👨‍🎓 Estudiante


📝 PASO 5: Crear tu Primera Cuenta
───────────────────────────────────────────────────────────────

OPCIÓN A: Registrar como Profesor
  1. Click en pestaña "Profesor"
  2. Click en "Registrarse"
  3. Completa el formulario:
     - Nombres, Apellidos
     - Correo (profesor@colegio.com)
     - Documento
     - Grado asignado (0-11)
     - Materia
     - Contraseña (mínimo 8 caracteres)
  4. Ingresa con tu correo y contraseña

OPCIÓN B: Registrar como Estudiante
  1. Click en pestaña "Estudiante"
  2. Click en "Registrarse"
  3. Completa el formulario:
     - Nombres, Apellidos
     - Número de documento
     - Grado (0-11)
     - Contraseña (mínimo 8 caracteres)
  4. Ingresa con tu documento y contraseña


📊 PORTAL DEL PROFESOR
───────────────────────────────────────────────────────────────

Al ingresar como profesor, verás:

  ✓ Tu información (Grado y Materia asignados)
  ✓ TABLA DE CALIFICACIONES:
    - Nombres y apellidos de estudiantes
    - Números de documento
    - Campos para ingresar notas (0-5)
    - Selector de trimestre (1-4)
    - Botón "Guardar Todas las Calificaciones"

  ✓ BANNER DE ASISTENTE IA:
    - Haz preguntas sobre pedagogía
    - Recibe respuestas de Gemini o OpenAI
    - Ejemplo: "¿Cómo enseñar fracciones en sexto?"


👨‍🎓 PORTAL DEL ESTUDIANTE
───────────────────────────────────────────────────────────────

Al ingresar como estudiante, verás:

  ✓ Tu información (Grado y Categoría)
  ✓ TABLA DE CALIFICACIONES:
    - Todas tus notas
    - Profesores que calificaron
    - Trimestre y período

  ✓ ASISTENTE IA:
    - Haz preguntas sobre temas escolares
    - Recibe respuestas inteligentes

🆘 PROBLEMAS COMUNES
───────────────────────────────────────────────────────────────

❌ "Cannot find module 'express'"
  ✓ Ejecuta: npm install

❌ "Port 3000 is already in use"
  ✓ Cambia PORT en .env a 3001, 3002, etc.

❌ "Database file not found"
  ✓ Se crea automáticamente, espera unos segundos

❌ "Invalid token"
  ✓ Vuelve a loguearte, el token expiró o no es válido
