📊 BASE DE DATOS (SQLite)

TABLAS:
  • students (estudiantes activos)
    - id, firstName, lastName, grade, gradeCategory
    - documentNumber, password, email, phone
  
  • teachers (profesores)
    - id, firstName, lastName, email
    - documentNumber, password, gradeAssigned, subject
  
  • grades (calificaciones)
    - id, studentId, teacherId, subject
    - score (0-5), quarter (1-4), period (año)
  
  • removed_students (borrado pasivo)
    - Copia de datos del estudiante al eliminarlo

🔌 ENDPOINTS PRINCIPALES

AUTENTICACIÓN:
  POST /api/auth/teacher/login          ← Login profesor
  POST /api/auth/student/login          ← Login estudiante
  POST /api/auth/teacher/register       ← Registrar profesor
  POST /api/auth/student/register       ← Registrar estudiante

ESTUDIANTES:
  GET    /api/students                  ← Obtener todos
  GET    /api/students/:id              ← Obtener uno
  POST   /api/students                  ← Crear
  PUT    /api/students/:id              ← Actualizar
  DELETE /api/students/:id              ← Eliminar (pasivo)

PROFESORES:
  GET    /api/teachers                  ← Obtener todos
  POST   /api/teachers                  ← Crear
  PUT    /api/teachers/:id              ← Actualizar
  DELETE /api/teachers/:id              ← Eliminar

CALIFICACIONES:
  GET    /api/grades/student/:id        ← Notas del estudiante
  GET    /api/grades/teacher/:id        ← Notas del profesor
  POST   /api/grades                    ← Crear calificación
  PUT    /api/grades/:id                ← Actualizar
  DELETE /api/grades/:id                ← Eliminar

ASISTENTE IA:
  POST   /api/assistant/ask             ← Consultar IA

🛠️ TECNOLOGÍA USADA

Backend:
  • Node.js + Express.js (servidor)
  • SQLite3 (base de datos)
  • JWT (autenticación)
  • Bcrypt (encriptación)

Frontend:
  • HTML5 (estructura)
  • CSS3 (estilos responsivos)
  • JavaScript Vanilla (interactividad)

IA:
  • Gemini API
  • OpenAI API
